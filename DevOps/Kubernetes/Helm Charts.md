**Helm** — это менеджер пакетов для [[K8s]]. Этот инструмент позволяет нам обернуть Kubernetes приложения в удобные пакеты, называемые чартами, которые можно легко развертывать, обновлять и управлять ими в любой момент времени.

Charts — это пакеты, которые могут включать в себя все для запуска приложения в Kubernetes, от deployments до services. Все это дает возможность работать с приложениями как с единой сущностью, а не как с набором отдельных ресурсов, которые еще и в ручную нужно настраивать.


**Полезные команды**:

- helm install <name> <chart> устанавливает чарт с специальным именем релиза
- helm uninstall <name> удаляет релиз 
- helm upgrade <release> <chart> одновляет релиз до нужной версии чарта или конфигурации
- helm rollback <release> <revision> переводит к обратной ревизии чарта с номером
- helm list показывает все релизы
- helm status <release> статус релиза 
- helm history <release> история релиза
- helm get values <release> скачивает файл values нужного релиза
- helm get manifest <release> показывает манифест ресурсов, установленных для релиза
- helm create <name> создает новую директорию чартов с нужным именем и структурой по умолчанию
- helm lint <chart> проверяет чарт на наличие потенциальных ошибок и не соблюдение лучших практик
- helm package <chart-path> собирает из директории версию чарта (в архив)
- helm repo add <name> <url> добавляет репозиторий чарта в конфигурацию Helm
- helm search repo <keyword> ищет репозиторий по слову
- helm search hub <keyword> ищет чарт в Helm Hub
- helm version выводит версию Helm client
- helm env выводит информацию по окружению
