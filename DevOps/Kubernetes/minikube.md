**Minikube** создает виртуальное окружение и ноды будут запускаться внутри него. То есть minikube это кластер K8s, состоящий из одной ноды, который запускается в Virtual Box и используется для обучающих и тестовых задач.

Для того, чтобы запустить minikube я сделал виртализацию с помощью **KVM (Kernel-based Virtual Machine)**. На самом деле KVM это просто драйвер в ядре Linux и при запуске ВМ Linux становится гипервизором, для самой ОС каждая ВМ это обычный процесс, но с доступом к виртуальному железу.

**Полезные команды:**
1. minikube start --driver=kvm2 - запускает minikube и в качестве драйвера использует KVM. При запуске в ВМ устанавливается Docker и компоненты K8s запускаются как докер контейнеры.
2. minikube status - для получения статуса можно воспользоваться командой.
3. minikube stop - останавливает minikube кластер.
