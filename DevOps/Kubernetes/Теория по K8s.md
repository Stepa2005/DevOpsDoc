Pod - самая маленькая единица в K8s, создается не pod, а [[Deployment]] (абстракция над pod).
При создании [[Deployment]],  он уже имеет всю информацию для создания подов. Самая базовая конфигурация для него это передать name и image, все остальное по умолчанию.
Между [[Deployment]] и Pod есть еще один уровень, которым автоматически управляет K8s - Replicaset. Это можно увидеть по названию:
![[Pasted image 20251208121612.png]] 
Replicaset управляет репликами pod, на практике это не используется, потому что вся работа происходит на уровне Deployment.

**Вот полный пирог абстракций: Deployment -> Replicaset -> Pod -> Container**. Все, что происходит ниже Deployment автоматически управляется кубернетисом.
