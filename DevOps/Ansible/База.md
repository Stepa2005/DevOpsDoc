Ansible (Automation Configuration Management Tool) - Система автоматизации конфигурации управления серверами. 

Для простого примера возьмем такую схему:
![[Pasted image 20260123220021.png]]
Для управления тремя серверами нужен один хост, на который и будет установлен Ansible (не надо устанавливать его на сервера).

В моем случае все элементы картинки будут представлять виртуальные машины, сделанные с помощью Oracle VM.

Для того, чтобы пингануть надо воспользоваться командой ``ansible all --key-file ~/.ssh/ansible/ -i inventory -m ping``, которая возьмет данные из inventory и пинганет те сервера,  IP/DNS которых там указан.

Файл конфигурации ansible.cfg будет прочитан ansible когда мы его запускаем.
